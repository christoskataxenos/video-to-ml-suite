# Project Guidelines: Papatzis Spoter

## 1. Vision & Purpose
Το `Papatzis Spoter` είναι ένα εργαλείο για την εξαγωγή frames από βίντεο.

## 2. Architecture
- **Engine Core (C++):** Decoding, resizing, filtering.
- **GUI (Python/CustomTkinter):** Wrapper για το C++ engine.

## 3. Coding Standards
### C++
- **Standard:** C++20.
- **Style:** Allman braces, 4-space indentation.
- **Comments:** Ελληνικά (# ή //).

### Python
- **Style:** 4-space indent, snake_case.
- **Comments:** Ελληνικά (#).
